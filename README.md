# cddatools

[CDDA Web Tools Directory](http://tools.cataclysmdda.com/) web application.

## Setup

```bash
$ git clone https://github.com/drbig/cddatools.git
$ cd cddatools
$ $EDITOR .env
$ bundle install
$ rake assets:all
$ foreman start
```
